Principais formatações

negrito / destaque

nessa frase tenhos um termo em negrito usando a tag B. (tag não semântica)

nessa frase temos um termo em destaque usando a tag STRONG (tag semântica)

Italico / Ênfase

Nessa frase temos um termo em italico usando a tag I (não semântico)

nessa frase temos um termo com ênfase usando a tag EM (semântica)

Texto marcado

Podemos criar um texto marcado usando a tag MARK

e no outro paragrafo temos outro texto marcado no final

texto pequeno

estamos criando um texto grande e um texto pequeno nesse paragrafo

texto deletado

Podemos marca um texto como excluido para indicar que ele deve ser lido, mas desconsiderado.

texto inserido

podemos marca um texto como inserido para dar uma ênfase e indicar que ele foi inserido depois.

também podemos usar o texto sublinhado usando a tag U (não semântico)

texto sobrescrito

Para inserir coisa do tipo x20+3 utilizamos a tag SUP

texto subscrito

para inserir coisas do tipo h2o utilizamos a tag SUB
